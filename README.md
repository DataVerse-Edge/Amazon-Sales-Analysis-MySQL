Project overview
This repository contains SQL solutions for an Amazon sales analysis assignment. The dataset (shared as a Google Sheet link in the assignment) includes product details such as name, category, actual price, discounted price, discount percentage, rating, and customer reviews. Your task is to query this data to extract actionable insights for management.

Questions to answer in SQL

Create SQL queries to solve the following tasks:

List all products with a discounted price below ₹500.​

Find products with a discount percentage of 50% or more.​

Retrieve all products where the name contains the word Cable.​

Display the difference between the average actual price and the average discounted price for each product.​

Query reviews that mention fast charging in their content.​

Identify products with a discount percentage between 20% and 40%.​

Find products that have an actual price above ₹1,000 and are rated 4 stars or above.​

Find products where the discounted price ends with a 9.​

Display review contents that contain negative‑sentiment words like worst, waste, poor, or not good.​

List all products where the category includes Accessories.

Notes
Adjust table and column names in the scripts to match your own schema.

This repository is intended for learning and practice; it does not include or redistribute the original dataset but assumes you have access to the shared Google Sheet.
